회사소개

서울시 공식 상권 데이터 기반 소상공인 분석 서비스

여기콕은 흩어져 있는 공공 상권 데이터를 지도 한 화면에 모아, 예비·기존 소상공인이 어디서 어떤 업종을 시작할지 데이터로 판단하도록 돕습니다.

무엇을 하나요

  • 지도 상권분석 · 서울 1,650여 개 상권의 매출·유동인구·점포수·성장/쇠퇴를 지도에서 바로 비교
  • AI 상권 리포트 · 선택한 상권의 지표를 모아 근거 기반 요약 리포트 생성
  • 업종·상권 동향 · 업종별 뉴스 인사이트, 상위 프랜차이즈 지표, 상표 출원 동향
  • 창업 지원사업 매칭 · 기업마당·K-Startup 지원사업을 업종·지역으로 연결

데이터 출처

공신력 있는 공공데이터만 사용합니다.

  • 서울 열린데이터광장 (매출·유동인구·상주인구·점포·집객시설·상권변화)
  • 한국부동산원 R-ONE (상업용 부동산 임대료·공실률)
  • 공정거래위원회 · 공공데이터포털 (프랜차이즈 브랜드·가맹현황)
  • 특허청 KIPRIS (상표 출원), 기업마당 · K-Startup (창업 지원사업)

이 서비스에 대해

여기콕은 대학 팀 프로젝트로 개발한 서비스입니다. 제공하는 분석은 공공데이터를 가공한 참고 정보이며, 창업·투자 결정의 최종 책임은 이용자에게 있습니다. 데이터 범위는 현재 서울시 상권을 기준으로 합니다.
